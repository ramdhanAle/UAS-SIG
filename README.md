# Peta Fasilitas STT Terpadu Nurul Fikri

Aplikasi Web GIS untuk memetakan fasilitas di lingkungan kampus STT Terpadu Nurul Fikri menggunakan Django dan Leaflet.

## Cara Menjalankan
1. Clone repo ini: `git clone ..`
2. Masuk ke folder: `cd ..`
3. Buat virtual environment: `python -m venv venv`
4. Aktifkan venv dan install requirement: `pip install -r requirements.txt`
5. Jalankan server: `python manage.py runserver`
